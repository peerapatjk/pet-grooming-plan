Pawpoint Customer · Expo Router implementation

The customer app now runs as a real Expo application.

This file is no longer the runtime surface. The static walkthrough has been retired in favor of a real Expo Router app with onboarding, discovery, booking, booking-state handling, visits, profile, and rating/rebook flows.

How to run it

  1. From prototype/claude-design/pet-grooming, run npm start.
  2. Launch the app on device or simulator with npm run ios or npm run android.
  3. Use npm run web only as an optional preview surface.
  4. The app entry is src/app/index.tsx; it routes into onboarding or the tab shell based on app state.
